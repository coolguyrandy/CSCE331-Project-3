Connect to Astra DB

Step 1

Make an Astra DB account -> AstraDB

Step 2

After you create your account you should see something like this; Click on where it says database


Step 3

Next you should see an option to create a database in the top right corner, Click that button and this should appear.


Fill in the respective fields and wait for your database to finish initializing

Step 4

Click on the database on the leftside. On the right side you should see your API endpoint as well as a way to generate a token

Step 5

Generate a token and then copy over both the token as well as the API endpoint into the text fields on the previous page.
